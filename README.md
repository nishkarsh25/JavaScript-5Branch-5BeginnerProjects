# Rocket Follow Mouse

Rocket Follow Mouse is a dynamic web project that allows a rocket image to smoothly follow the movement of the mouse cursor on a webpage. This README provides comprehensive information on the project, including its features, how to run it locally, licensing, and how to contribute.

## Features

### Mouse Cursor Tracking
The rocket image elegantly tracks the movement of the mouse cursor in real-time, creating an engaging and interactive user experience.

### Responsive Design
The project is designed to be responsive, ensuring that it adapts seamlessly to various screen sizes and devices, including desktops, laptops, tablets, and smartphones.

### Tailwind CSS Styling
Tailwind CSS, a highly customizable utility-first CSS framework, is used to style the elements in the project. This framework provides a modern and visually appealing appearance while maintaining flexibility and scalability.

## Usage

Once the project is running in your web browser, simply move your mouse cursor around the webpage, and you'll observe the rocket image smoothly following its movement. Enjoy the interactive experience!
